frank Brouwer-Heyting-Kolmogorov interpretation of intuitionistic logic
suggests that p implies q can be interpreted as a computation that

    intuitionistic logic, then two worlds in K are bisimilar if and
    only if franky satisfy the same set of formulas.

    and only if franky have the same set of valid formulas.
